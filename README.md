# todo-react-app

Todo app is created at React redux-toolkit and tailwindcss

## Installation

Use the package manager [npm](https://nodejs.org/en/) or [yarn](https://classic.yarnpkg.com/lang/en/docs/install/) to install todo-react-app.

```bash
npm install
```
```bash
yarn install
```

## development

```bash
npm run dev
```
```bash
yarn dev
```
